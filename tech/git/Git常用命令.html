Git常用命令

GIT

专有名词说明

  1. 版本库:.git文件夹就是整个版本库
  2. 暂存区:.git文件夹里面有很多数据，其中比较重要的是暂存区，分支，历史仓库
  3. 工作区:项目中除.git文件夹之外的就是工作区

变量配置

用户名和邮件地址

  1. git config --global user.name "yuyejiang"
  2. git config --global user.email "980201346@qq.com"

别名设置

  1. git config --global alias.st status
  2. git config --global alias.ci commit
  3. git config --global alias.co checkout
  4. git config --global alias.br branch

Log显示格式设置

  1. git config --global alias.glog "log --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr)%Creset %cn' --abbrev-commit --date=relative"

Git 初始化

本地创建仓库并关联远程仓库

  1. git init 创建本地仓库
  2. git add . 本地仓库中的文件全部添加到暂存区
  3. git commit -m "初始化项目" 提交
  4. git remote add <远程url别名，自己定义> <url>
  5. git push <远程url别名，自己定义> <local branch name>
  6. git push --set-upstream <远程url别名，自己定义> <远程分支名称> 设置本地分支push时提交到服务器的默认分支

直接从远程克隆仓库

  1. git clone <远程仓库地址>
  2. git pull <远程仓库名称> <远程分支名>:<本地分支名> 命令中:<本地分支名>可以忽略，忽略之后默认代码合并到当前分支
